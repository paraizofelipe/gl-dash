---
title: Updating
---

You can update `dash` by downloading the latest binary or by rebuilding it from source.

## Update Manually

To manually update `dash`, download the latest binary from the releases page under:
https://gitlab.com/GROUP/gl-dash/-/releases

Search for the binary that corresponds to your operating system and CPU
architecture, then replace your existing installed binary with it.

## Update From Source

If you built `dash` from source, pull the latest changes and rebuild:

```sh
git pull
go build -o gl-dash .
```

Get started using `dash` to review and manage your GitLab work items.

## Recommended Steps

1. Download the latest binary for your platform from the [releases page][01]:

   ```bash
   curl -fsSL "https://gitlab.com/GROUP/gl-dash/-/releases/permalink/latest/downloads/gl-dash_$(go env GOOS)-$(go env GOARCH)" -o gl-dash
   chmod +x gl-dash
   sudo mv gl-dash /usr/local/bin/gl-dash
   ```

2. To get the icons to render properly, you should download and install a [Nerd font][02]. Then,
   select that font as your font for your terminal of choice.

   ```bash
    brew install --cask font-fira-code-nerd-font
   ```

## Building From Source

If you'd rather build `gl-dash` yourself, follow these steps:

1. Clone the repo

   ```bash
   git clone https://gitlab.com/GROUP/gl-dash.git
   ```

2. `cd` into it

   ```bash
   cd gl-dash
   ```

3. Build it locally

   ```bash
   go build -o gl-dash .
   ```

Coming from `gh-dash`? See [Migrating from gh-dash][03].
